# 📊 E-Commerce Data Analysis & Dashboard

## 📌 Project Overview
This project focuses on analyzing E-commerce sales data to extract meaningful insights and support data-driven decision-making. It includes data cleaning, exploratory data analysis (EDA), and an interactive Excel dashboard.

---

## 🎯 Objectives
- Analyze sales performance across categories, regions, and products  
- Identify key business trends and patterns  
- Build an interactive dashboard for better visualization  
- Generate actionable insights for decision-making  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)  
- Data Visualization (Matplotlib, Seaborn)  
- Microsoft Excel (Pivot Tables, Charts, Slicers)  

---

## 📂 Dataset
- Synthetic E-commerce dataset  
- Contains order details, sales, profit, quantity, category, region, and product information  

---

## 📊 Excel Dashboard Features
- KPI Metrics:
  - Total Sales  
  - Total Profit  
  - Total Orders  
  - Total Quantity  
  - Average Order Value (AOV)  

- Visualizations:
  - Sales by Category  
  - Sales by Region  
  - Monthly Sales Trend  
  - Top Products Analysis  

- Interactive Filters:
  - Category  
  - Region  
  - Date  

---

## 📈 Key Insights
- Electronics category contributes the highest revenue  
- Certain regions show stronger sales performance  
- Top products drive a significant portion of total sales  
- Sales trends vary across months, indicating seasonal patterns  

---

## 📁 Project Structure
- `ecomm_1.ipynb` → Data analysis using Python  
- `ecommerce_returns_synthetic_data.csv` → Dataset  
- `Ecommerce_Dashboard.xlsx` → Excel dashboard  

---

## 🚀 How to Use
1. Open the Jupyter Notebook to explore data analysis  
2. Review the dataset for raw data  
3. Open the Excel dashboard to interact with visual insights
